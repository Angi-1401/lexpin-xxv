CSS: inline, internal, and external styles

Short explanation and live examples showing how to apply CSS using three different methods.

1. Inline CSS

Inline styles are applied directly on an element using the style attribute. They have the highest specificity for that element.

This box is styled with inline CSS. 

<div style="border: 4px dashed rebeccapurple; padding: 16px;">
  <p style="color: rebeccapurple;">
    This box is styled with inline CSS.
  </p>
</div>

2. Internal (Embedded) CSS

Internal CSS is placed inside a <style> tag in the document head. It's useful for page-specific styles without creating a separate file.

This box is styled using internal (embedded) CSS. 

<!-- inside <head> -->
<style>
.internal-box {
  border: 8px solid coral;
  padding: 20px;
  background: #fff7f6;
}
</style>

<!-- in body -->
<div class="internal-box">…</div>

3. External CSS

External styles live in a separate .css file and are linked with <link rel="stylesheet" href="..."> . This is the recommended approach for site-wide styles and better caching.

This box is styled using the external stylesheet
( ./css/styles.css ). 

<!-- in <head> -->
<link rel="stylesheet" href="./css/styles.css">

<!-- in body -->
<div class="box">…</div>

Notes & tips

  • Inline styles are convenient for quick overrides but are harder to maintain and don't separate concerns.
  • Internal styles are good for single-page overrides or demos, but prefer external files for multi-page sites.
  • External stylesheets provide reuse, caching, and cleaner HTML structure.
